# 画面設計

設計判断（`docs/working/design/04-screen-design.md`）をもとに整理。

## 技術スタック

| 項目 | 選定 |
|------|------|
| フレームワーク | Next.js (App Router) |
| UI ライブラリ | shadcn/ui + Tailwind CSS |
| サーバーステート | TanStack Query (React Query) |
| フォーム | React Hook Form + Zod |
| レイアウト | サイドバー + メインコンテンツ |

---

## ページ一覧

### 認証

| パス | ページ名 | 概要 | 権限 |
|------|---------|------|------|
| `/login` | ログイン | メール・パスワード入力 | 未認証 |

### 一般社員（全ロール共通）

| パス | ページ名 | 概要 | 対応ストーリー |
|------|---------|------|--------------|
| `/` | ダッシュボード（打刻） | 出勤/退勤ボタン、当日の打刻状態 | ATT-01〜06 |
| `/history` | 勤怠履歴 | 月別の自分の勤怠一覧、勤務時間表示 | HIST-01〜04 |
| `/corrections` | 修正申請一覧 | 自分の修正申請の状態確認 | FIX-03 |
| `/corrections/new` | 修正申請作成 | 修正内容と理由を入力して申請 | FIX-01, FIX-02 |

### 上長

| パス | ページ名 | 概要 | 対応ストーリー |
|------|---------|------|--------------|
| `/team` | 部署メンバー勤怠 | 自部署メンバーの月別勤怠一覧 | HIST-05 |
| `/approvals` | 承認待ち一覧 | 修正申請の承認/却下 | FIX-04, FIX-05 |

### 管理者

| パス | ページ名 | 概要 | 対応ストーリー |
|------|---------|------|--------------|
| `/admin/employees` | 社員管理 | 社員一覧・登録・編集・退職・上長設定 | EMP-01〜06 |
| `/admin/departments` | 部署管理 | 部署一覧・登録・編集 | DEPT-01, DEPT-02 |
| `/admin/attendance` | 全社員勤怠 | 全社員の勤怠一覧（部署フィルタ付き） | HIST-06 |
| `/admin/reports` | 月次集計 | 月次集計表示・CSV/PDF エクスポート | RPT-01〜03 |

## サイドバーメニュー

ログインユーザーのロール・上長フラグに応じてメニュー項目を出し分ける。

```
全ロール共通:
  📋 打刻            → /
  📅 勤怠履歴        → /history
  ✏️ 修正申請        → /corrections

上長のみ:
  👥 部署メンバー勤怠 → /team
  ✅ 承認待ち        → /approvals

管理者のみ:
  👤 社員管理        → /admin/employees
  🏢 部署管理        → /admin/departments
  📊 全社員勤怠      → /admin/attendance
  📈 月次集計        → /admin/reports
```

### ダッシュボード / 打刻 (`/`)

```
┌──────┬──────────────────────────────────┐
│      │  こんにちは、田中太郎さん          │
│ サイ  │                                  │
│ ド   │  2026年6月17日（火）               │
│ バー  │  現在のステータス: 出勤中           │
│      │                                  │
│      │  ┌──────────┐  ┌──────────┐      │
│      │  │  出勤     │  │  退勤     │      │
│      │  │ (disabled)│  │          │      │
│      │  └──────────┘  └──────────┘      │
│      │                                  │
│      │  ── 本日の打刻記録 ──              │
│      │  #1  09:00 出勤  →  まだ          │
│      │                                  │
└──────┴──────────────────────────────────┘
```

コンポーネント:
- `ClockButtons` — 出勤/退勤ボタン。ステータスに応じて活性/非活性を切り替え
- `TodayRecords` — 当日の打刻記録リスト
- `StatusBadge` — 現在のステータス表示（NOT_CLOCKED_IN / CLOCKED_IN / CLOCKED_OUT）

状態遷移:
- 未出勤 → 出勤ボタンのみ活性
- 出勤中 → 退勤ボタンのみ活性
- 退勤済み → 出勤ボタン活性（再出勤）

### 修正申請作成 (`/corrections/new`)

```
┌──────┬──────────────────────────────────┐
│      │  修正申請                         │
│ サイ  │                                  │
│ ド   │  対象日:  [2026-06-14]            │
│ バー  │                                  │
│      │  対象レコード: [選択 ▼]  ※任意     │
│      │  （打刻忘れの場合は空欄）           │
│      │                                  │
│      │  修正後の出勤時刻: [09:00]         │
│      │  修正後の退勤時刻: [18:00]         │
│      │                                  │
│      │  理由:                            │
│      │  ┌───────────────────────────┐    │
│      │  │ 打刻を忘れていたため       │    │
│      │  └───────────────────────────┘    │
│      │                                  │
│      │  [キャンセル]  [申請する]           │
└──────┴──────────────────────────────────┘
```

コンポーネント:
- `CorrectionForm` — 修正申請フォーム
  - 対象日を選択 → その日の既存打刻レコードを取得して選択肢に表示
  - 既存レコードを選んだ場合: 現在の値をプリフィル
  - レコードなし（打刻忘れ）: 空欄で入力
- バリデーション: 出勤時刻 < 退勤時刻、理由は必須

## 共通コンポーネント

| コンポーネント | 説明 | 使用画面 |
|--------------|------|---------|
| `AppLayout` | サイドバー + ヘッダー + メインコンテンツのレイアウト | 全画面（ログイン除く） |
| `Sidebar` | ロールに応じたナビゲーションメニュー | 全画面（ログイン除く） |
| `Header` | ユーザー名 + ログアウトボタン | 全画面（ログイン除く） |
| `MonthSelector` | 月の前後切り替え | 履歴・チーム勤怠・集計 |
| `StatusBadge` | ステータスのバッジ表示（色分け） | 打刻・修正申請 |
| `DataTable` | shadcn/ui ベースの共通テーブル（ソート・ページネーション） | 各一覧画面 |
| `FormDialog` | shadcn/ui Dialog ベースのフォームモーダル | 登録・編集系 |
| `ConfirmDialog` | 確認ダイアログ（承認・却下・退職処理等） | 操作確認系 |
| `Toast` | 操作結果の通知（成功・エラー） | 全画面 |
