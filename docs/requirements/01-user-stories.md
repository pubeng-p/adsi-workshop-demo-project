## ロール定義

| ロール | 説明 | 備考 |
| --- | --- | --- |
| 一般社員 | 一般の従業員 | 自分の勤怠のみ操作可能 |
| 上長 | 部署の課長・部長相当 | 部署ごとの上長フラグで管理。一般社員の権限も持つ |
| 管理者 | 人事部門の担当者 | 社員マスタ管理・月次集計を担当。一般社員/上長とは別軸のロール |

- 管理者ロールと上長フラグの兼任は可能
- 管理者も任意の部署に所属し、打刻・勤怠修正の文脈では一般社員/上長と同じルールに従う
- 上長不在時の代行は対象外（デモスコープ）

## 打刻

| ID | ストーリー | ロール |
| --- | --- | --- |
| ATT-01 | 出勤ボタンを押して出勤を打刻できる | 全ロール |
| ATT-02 | 退勤ボタンを押して退勤を打刻できる | 全ロール |
| ATT-03 | 当日の打刻状態（未出勤 / 出勤済み / 退勤済み）を確認できる | 全ロール |
| ATT-04 | 出勤中（未退勤）に再度出勤打刻はできない（エラーになる） | 全ロール |
| ATT-05 | 出勤せずに退勤打刻はできない（エラーになる） | 全ロール |
| ATT-06 | 同日に退勤後、再度出勤→退勤ができる（複数回打刻） | 全ロール |

- 管理者も一般社員と同様に打刻する

## 勤怠履歴

| ID | ストーリー | ロール |
| --- | --- | --- |
| HIST-01 | 自分の勤怠履歴を日別で確認できる | 全ロール |
| HIST-02 | 自分の勤怠履歴を月別で確認できる | 全ロール |
| HIST-03 | 勤務時間・残業時間が自動計算されて表示される | 全ロール |
| HIST-04 | 修正された勤怠データには修正済みマークが表示される | 全ロール |
| HIST-05 | 自部署メンバーの勤怠履歴を確認できる | 上長 |
| HIST-06 | 全社員の勤怠履歴を確認できる | 管理者 |

- 修正前の元の値（修正履歴）の参照は不要。修正済みマークのみ

## 勤怠修正

| ID | ストーリー | ロール |
| --- | --- | --- |
| FIX-01 | 打刻忘れや誤りの修正を申請できる | 全ロール |
| FIX-02 | 修正申請には修正後の出勤/退勤時刻と理由を入力する | 全ロール |
| FIX-03 | 自分が出した修正申請の状態（申請中/承認/却下）を確認できる | 全ロール |
| FIX-04 | 自部署メンバーの修正申請を一覧で確認できる | 上長 |
| FIX-05 | 自部署メンバーの修正申請を承認または却下できる | 上長 |
| FIX-06 | 承認された修正は自動的に勤怠データに反映され、修正済みマークが付く | システム |
| FIX-07 | 承認操作と取り下げ操作が競合した場合、後の操作が楽観ロックエラーになる | システム |

### 承認フロー

| 申請者 | 承認者 |
| --- | --- |
| 一般社員 | 所属部署の上長 |
| 上長 | 自己承認 |
| 管理者（上長でない） | 所属部署の上長 |
| 管理者（上長を兼任） | 自己承認 |

## 勤務時間計算

| ID | ルール |
| --- | --- |
| CALC-01 | 同日複数回打刻の場合、各回の勤務時間を合算する: Σ(退勤n − 出勤n) |
| CALC-02 | 休憩控除: 合算勤務時間が 6h 超〜8h 以下 → 45 分、8h 超 → 60 分、6h 以下 → 0 分（労基法準拠） |
| CALC-03 | 勤務時間 = 合算勤務時間 − 休憩控除時間 |
| CALC-04 | 残業時間 = 勤務時間（休憩控除後）− 8 時間。マイナスの場合は 0 |

## 社員マスタ管理

| ID | ストーリー | ロール |
| --- | --- | --- |
| EMP-01 | 社員を新規登録できる（氏名・メール・パスワード・部署・ロール・入社日） | 管理者 |
| EMP-02 | 社員情報を編集できる | 管理者 |
| EMP-03 | 社員を退職扱いにできる（退職日を設定。論理削除） | 管理者 |

| EMP-06 | 退職した社員の過去の勤怠データを参照できる | 管理者 |

### 社員マスタの項目

| 項目 | 必須 | 備考 |
| --- | --- | --- |
| 社員 ID | 自動採番 | UUID |
| 氏名 | ○ | |
| メールアドレス | ○ | ログイン ID として使用。一意制約 |
| パスワード | ○ | BCrypt ハッシュ化。変更・リセット機能はスコープ外 |
| 所属部署 | ○ | 部署テーブルへの FK |
| ロール | ○ | 一般社員 / 管理者 |
| 上長フラグ | ○ | その部署の上長かどうか。デフォルト false |
| 入社日 | ○ | |
| 退職日 | − | null = 在籍中。設定するとログイン不可 |

## 月次集計

| ID | ストーリー | ロール |
| --- | --- | --- |
| RPT-01 | 指定月の全社員の勤怠集計を画面で確認できる | 管理者 |
| RPT-02 | 月次集計を CSV でエクスポートできる | 管理者 |
| RPT-03 | 月次集計を PDF 帳票として出力できる（JasperReports） | 管理者 |

### 月次集計に含まれる項目

- 社員名
- 所属部署
- 出勤日数
- 勤務時間合計
- 残業時間合計
- 欠勤日数（営業日 = 平日（土日除く）に出勤打刻がない日）

## 初期データ

| ID | ストーリー | ロール |
| --- | --- | --- |
| SEED-01 | Flyway マイグレーションでデモ用の初期データが投入される | システム |

初期データの内容:

- 部署: 3〜4 部署（例: 人事部、開発部、営業部、総務部）
- 社員: 各部署に 2〜3 名 + 管理者 1 名 + 各部署に上長 1 名
- 勤怠データ: 直近 1 ヶ月分のサンプルデータ
- パスワード: 全アカウント共通の固定値（デモ用）
